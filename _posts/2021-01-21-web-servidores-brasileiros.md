# Web: Servidores brasileiros

Nuvem... e servidores DNS no Brasil: conheça!

<h5>Nuvem?</h5>
É um termo utilizado para descrever uma rede global de servidores.

Operando como um único ecossistema, a "nuvem" é, na verdade, uma grande rede de servidores remotos ao redor do globo que são conectados.

A Microsoft tem datacenter no Brasil. Já a Google possui o servidor GCP (Google Cloud Platform). A Amazon Web Services (AWS) é uma subsidiária da Amazon.
