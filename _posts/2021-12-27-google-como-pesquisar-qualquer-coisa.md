# Google: como pesquisar qualquer coisa

Conheça truques para descobrir qualquer coisa na Web.



Imagens, vídeos e outros tipos de conteúdo são bem comuns na Web.

Mas são os textos, que desde a invenção da escrita até hoje, guardam conhecimentos preciosos. E é através dele, ou melhor palavras-chaves e <em>"comandinhos"</em>, que podemos 'nos comunicar' com o Google e obter uma pesquisa melhorada.

Lidando bem com o texto e a informática podemos achar outros textos; da mesma forma vídeos, arquivos do tipo pdf, etc. Saber fazer isso é algo considerável, e somado a boas fontes de informação são dois diferenciais.
Neste caso das fontes veja <a href="https://joesynth.wordpress.com/2021/11/06/onde-pesquisar-no-google-errou/">Onde Pesquisar? No google? Errou!</a>.

Vamos lá?
<h5>Pesquisa Google</h5>
<strong>Preposição</strong>: uma pesquisa com ou sem preposição, "de" por exemplo, nos traz resultados diferentes. Feita a pesquisa, observe em <em>"aproximadamente... resulados"</em> e note a diferença de números para ambos os casos.
<pre>quadro de madeira // Retorna "Aproximadamente 34mi resultados"
quadro madeira // Retorna "Aproximadamente 33mi resultados"</pre>
⚠️ A mesma ideia vale para artigo.

<strong>Busca exata</strong>: palavras na ordem exata. Aqui o Google trará menos resultados (e não várias combinações).
<pre>"como jogar bola"
</pre>
Note que, com ou sem aspas, muda a quantidade de resultados retornados pelo Google.
<pre>pedro paulo // Retorna "Aproximadamente 250mi resultados"
"pedro paulo" // Retorna "Aproximadamente 6mi resultados"</pre>
<strong>Curinga</strong> (asterísco): completa a palavra.  Abaixo, retorna presidente Bolsonaro.
<pre>presidente bol*
</pre>
<strong>Busca com exclusão</strong> (espaçoMenos): neste caso retorna nada que tenha 'basquete'. Cuidado! É <strong>espaço menos</strong>.
<pre>esporte -basquete
</pre>
<strong>Combinação?</strong> Também é possível usar o truque das aspas e do menos, e assim filtrando ainda mais.
<pre>"esporte com bola" -basquete</pre>
<strong>Mostra arquivo específico</strong>: comando "filetype:". Teremos aqui o retorno de arquivos <em>pdf</em> sobre <a href="https://joesynth.wordpress.com/2021/11/28/por-que-usar-linux/">Ubuntu</a>.
<pre>ubuntu filetype:pdf
</pre>
<strong>Pesquisa duas coisas juntas</strong>: Comando lógico OR (ou).
<pre>vasco OR flamengo</pre>
Note que o resultado seria diferente, caso digitássemos 'vasco flamengo'; sem OR.
<h5>Outros comandos..</h5>
Mais comandos? Alguns destes farão mais sentido se você tiver noção de HTML.

Páginas que apontam para um site
<pre>link:www.joesynth.wordpress.com.</pre>
Palavra aparece no <em>link</em>. Comandinho "allinanchor:"
<pre>allinanchor:vsti</pre>
Palavra aparece no título. Comandinho "allintitle:"
<pre>allintitle:vsti</pre>
Procura específica dentro de um site. Um assunto, tema...
<pre>sintetizador site:joesynth.wordpress.com
</pre>
As páginas dentro de um site?  Comandinho "site:". Vai retornar (escrito no Google) <em>"aproximadamente... resultados"</em>.
<pre>site:joesynth.wordpress.com 
</pre>
Busca dentro de texto: ou seja, pelo conteúdo dos arquivos que estão nos servidores. Aqui o comando pesquisa conteúdo do site mais conteúdo dos arquivos do site; com isso, estamos procurando exatamente uma palavra dentro de um arquivo tal.
<pre>intext:harry porter</pre>
Procura pelo título do arquivo HTML, a <em>tag</em> HTML.
<pre>intitle:</pre>
Procura dentro do URL.
<pre>inurl:</pre>
Palavra pesquisada aparece no título: operador +. Semelhante ao menos do exemplo citado, porém em vez de excluir vai incluir.
<pre>rede + computadores 
</pre>
