
O que é CLI? E GUI?

Conheça outra forma de interagir com o computador, além do <em>mouse</em>.



<span style="color: #0000ff"><em>Command Line Interface</em>, Interface de Linha de Comando</span>.

Ele é um programa que permite o usuário digitar comandos em texto e dar instruções ao computador para fazer algo específico.

<img class="alignleft" src="https://upload.wikimedia.org/wikipedia/commons/thumb/2/29/Linux_command-line._Bash._GNOME_Terminal._screenshot.png/300px-Linux_command-line._Bash._GNOME_Terminal._screenshot.png" />Provavelmente você conhece a CLI graças às telas pretas com textos em verde (ou na cor branca) e que vemos em filmes, quando programadores ou <em>hackers</em> estão trabalhando.

Poderosa, é uma ferramenta bem conhecida de desenvolvedores, e não se limita só a um usuário avançado em computação. Você mesmo pode usar! E fique esperto: para algumas tarefas dedicadas temos um rendimento melhor.

Em suma os sistemas operacionais têm uma interface de linha de comando para acesso às funções (ou serviços) do próprio <a href="https://joesynth.wordpress.com/2021/11/27/sistemas-operacionais-mais-usados/">SO</a> (sistema operacional).
<h6>Shell <strong><span style="color: #0000ff">concha</span></strong></h6>
Termo comum para sistemas operacionais <a href="https://joesynth.wordpress.com/2021/11/27/sistemas-operacionais-mais-usados/">Unix</a>, se referi a camada externa entre o usuário e o núcleo (<em>kernel</em>) de um sistema operacional.

<em>Shell: "...é o componente do sistema que fornece a interface em modo texto, convertendo os comandos dados pelo usuário nas instruções entendidas pelo kernel do sistema".</em>
<span style="color: #808080"><em>Morimoto</em></span>.

Em suma, o <em>shell</em> "<em>cuida da interação do sistema operacional com o usuário</em>". <span style="color: #808080">Dicionário dos Usuários de Microcomputadores</span>.
<h6>CLIs populares</h6>
Sistemas operacionais, como o <a href="https://joesynth.wordpress.com/2021/11/27/sistemas-operacionais-mais-usados/">Windows</a> ou <a href="https://joesynth.wordpress.com/2021/11/27/sistemas-operacionais-mais-usados/">Linux</a>, usam a CLI; sendo que há duas bem comuns:

<strong>DOS</strong> (<em>Disk Operating Systems</em>; Windows): o <em>shell</em> padrão no Windows é o CMD.exe ou o Prompt de Comando (<em>Command Prompt</em>).
<strong>Bash</strong> (<em>Bourne Again Shell</em>, Free Software Foundation): tipo de <em>shell</em> usado no Linux e macOS. No Linux é conhecido por <em>Terminal</em> (Gnome) ou <em>Konsole</em> (KDE).

😜️ Embora utilizem sintaxes de comandos diferentes eles têm uma maneira de operar até parecida.
<h6>Software de linha de comando</h6>
Não existe só o DOS e o Bash que lidam com CLI. Há por exemplo softs de áudio e vídeo, dentre outros. Veja <a href="https://joesynth.wordpress.com/2021/12/07/software-de-linha-de-comando/">software de linha de comando</a>.
<h1>CLI e GUI</h1>
Fique <em>esperto</em> nesses dois tipos de interface.
<h5>CLI</h5>
<em>Command Line Interface</em>, Interface de Linha de Comando.

Basicamente, aqui vamos "conversar" com o computador utilizando o teclado, ao invés do <em>mouse</em>.

Veja o artigo <a href="///home/joe/appjoB/o-que-é-cli.html"> O que é CLI?</a> e entenda onde e como é usado.
<h5>GUI</h5>
Graphical User Interface - Interface gráfica do usuário.

É uma forma de interface do usuário que permite você (o usuário) lidar com dispositivos eletrônicos através de informações organizadas em ícones, janelas.
Nela o ator principal é o <em>mouse</em> (diferente da CLI, onde o <em>cara</em> mais importante é o teclado).

Sem dúvida a GUI é mais atraente visualmente, comparada à CLI (que também tem suas vantagens: leia <a href="///home/joe/appjoB/o-que-é-cli.html">O que é CLI?</a>).
<h5>GUI: alguns termos</h5>
<strong>Radio</strong> (ou <em>option button</em>, <em>radio button</em>) Botão de opção usado em grupo de dois ou mais e ao selecionarmos um desabilita-se o outro.
<strong>Toggle</strong> (<em>switch</em>=interruptor, comutador) Ao ligar o visual muda, parecendo ligado.
<strong>Widget</strong> É um componente de uma GUI. Também se refere a um pequeno app que flutua na área de trabalho e dá função específica como ao usuário (relógio, cotação de moeda ) . Alguns visam receber dados do usuário e assim gerar certo tipo de rgistro.

Vale lembrar que optar em usar a GUI ou a CLI é uma escolha sua!
<h3>Um pouco de história</h3>
<h5>CLI vs GUI: origens</h5>
<img class="alignleft" src="https://upload.wikimedia.org/wikipedia/commons/thumb/0/0e/Hard_reset_BBC_Micro_32K_Acorn_DFS.gif/220px-Hard_reset_BBC_Micro_32K_Acorn_DFS.gif" width="150" /> 🇬🇧 <em>BBC, início anos 80</em>.
Lá nos anos 60 a CLI era bem usada. Nessa época uma pessoa só tinha o teclado como dispositivo de entrada, e a tela do computador mostrava informações em texto; tal como nesse BBC da foto. Pra fazer algo o usuário digitava um comando na CLI, e assim ia se comunicando com o computador.

<img class="alignleft" src="https://upload.wikimedia.org/wikipedia/en/thumb/1/1d/Xerox_Star_8010_workstations.jpg/220px-Xerox_Star_8010_workstations.jpg" width="150" />🇺🇲️ <em>Xerox Star, início anos 80</em>.
A GUI se desenvolveu dentro dos sistemas operacionais assim que o <em>mouse</em> se tornou o dispositivo de  <em>point-and-click</em> (apontar e clicar) em um computador. Ou seja, um novo jeito de interagir com o computador.

<img class="size-full wp-image-3552 alignleft" src="https://joesynth.files.wordpress.com/2019/02/amiga_atari_mac.png" alt="logos dos computadores Amiga, Atari, Mac." width="150" /> Até o final da década de 80, computadores como o Atari, Macintosh e o Amiga utilizaram naturalmente esse estilo de interface com ícones e <em>mouse</em>. Aliás, esses aí viraram sinônimos de computador multimídia, sendo inclusive utilizados em estúdios musicais profissionais.
