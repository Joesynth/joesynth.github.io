<h1 id="title">JSON</h1>
<span style="color:#0000ff">JavaScript Object Notation</span>.

<img class="alignleft" src="https://www.w3schools.com/js/img_json.jpg" />

Douglas Crockford []
json.org

JSON é um formato de dados independente da linguagem. JSON é uma variação da JS; subconjunto da JS. Tecnologia alternativa ao XML; ou seja, substitui o XML (pois se torna mais agil devido a certos fatores). É um formato leve. Permite intercâmbio de dados entre cliente e servidor. Ajuda a criar variáveis complexas de maneira simples. Tamanho reduzido. Seu uso não requer JS exclusivamente. Pode ser interpretado em várias linguagens. Formato de dados para comunicação assíncrona navegador/servidor, AJAX...

É um construto de Notação.
Construto é algo que construímos.
Notação é qualquer conjunto de símbolos e abreviações que nos ajudam a trabalhar em um determinado assunto.

<strong>Extensão</strong> .json
<strong>Soft Editor</strong> Qualquer editor de texto.
<strong>Derivado de...</strong> Deriva da JS.

Código (Exemplo)
<pre> { "alunos" : [
     {"nome": "joão", "notas" : [ 8, 7 ] },
     {"nome": "pedro", "notas" : [ 7, 9 ] }
     ]
 }</pre>
<h2>Sintaxe</h2>
Dado JSON é escrito como pares nome/valor.
<pre>"nome":"Jesus"
</pre>
<blockquote>Os nomes JSON requerem aspas duplas. Nomes JavaScript não.</blockquote>
JS
<pre>nome:"Jesus"
</pre>

<pre>string

<h3>JSON em forma de objeto de vetores</h3>
<pre>var array = [
 {"nome": [0,1,2], "telefone": "999-777-666", "cidade": "Niteroi"},
 {"nome": "Ultra", "telefone": "999-666-555", "cidade": ["Nova Iguaçu", "Nilópolis"]}
];
document.write("O índice array[0].nome[1] do array:" + array[0].nome[1]);
// imprime &gt;&gt; O índice array[0].nome[1] do array: 1

document.write("O índice array[1].cidade[0] do array:" + array[1].cidade[0]);
</pre>
<h3>JSON em forma de vetor de objetos</h3>
<pre>var array = [
 {"nome": "Gigaweb", "telefone": "999-777-666", "cidade": "Niteroi"},
 {"nome": "Ultra", "telefone": "999-666-555", "cidade": "Nova Iguaçu"}
];
document.write("O índice 0.nome do array:" + array[0].nome);
// imprime &gt;&gt; O índice 0.nome do array: Gigaweb

document.write("O índice array[1].cidade[0] do array:" + array[1].cidade[0]);
</pre>
